# Demo media checklist

This directory is reserved for presentation assets referenced by the root `README.md`.

Expected files:

- `ur5e_obstacle_demo.gif`: short animated GIF for README preview.
- `gazebo_obstacle_scene.png`: Gazebo screenshot showing the robot, table, cube obstacle, and target object.
- `rviz_motion_plan.png`: RViz screenshot showing the MoveIt planned trajectory.

Recommended 30-60 second video structure:

1. Show Gazebo + RViz running together.
2. Show the obstacle scene: table, cube obstacle, target object, narrow passage, or mid-arm obstacle.
3. Run one planning/execution demo.
4. Show benchmark plots and briefly explain success rate, planning time, tracking error, response time, and overshoot.

After uploading the video, replace the placeholder row in the root `README.md` with the final Bilibili, YouTube, or cloud-drive link.
